VPN encrypts traffic using:

AES-256 encryption

Secure tunneling protocol 

This prevents:

ISP monitoring browsing activity

Public Wi-Fi packet sniffing

Man-in-the-middle attacks

Benefits of using VPN

Hides real IP address

Encrypts internet traffic

Protects data on public Wi-Fi

Bypasses geo-restrictions

Reduces ISP tracking

Limitations of using a VPN 

Slower speeds

VPN provider can technically see traffic metadata

Does NOT make you anonymous

Does NOT protect against malware

Free plans have restrictions
